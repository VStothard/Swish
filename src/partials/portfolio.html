A blog site that uses Foundation for it's front end framework and Contentful as the CMS / back end.

This was an experimental project testing the usage and maintainability of a headless CMS. Over all my experience with Contentful has been very positive, it is a clean and easy to use platform, great for quickly implementing a CMS customised to your particular use case.
